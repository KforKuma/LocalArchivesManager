# LAM public test corpus

This corpus contains only project-generated synthetic PDFs and small citation
text fixtures. It contains no user Catalogue, library state, topic notes, or
third-party article PDF.

- `included/`: redistributable synthetic PDFs generated by the project.
- `reference_text/`: UTF-8/LF citation and title-list fixtures.
- `expected/`: reserved for stable golden outputs; expectations currently live
  in `manifest.json`.
- `downloaded/`: ignored local destination for explicitly `download_only`
  cases.
- `manifest.schema.json`: JSON Schema for the public manifest contract.

Regenerate the PDF fixtures from the source root:

```powershell
python scripts/generate_test_corpus.py
```

Fetch configured external cases without overwriting local files:

```powershell
python scripts/fetch_test_corpus.py
```

The repository currently configures no external download-only paper. A case
must have a stable direct HTTP(S) URL, a verified SHA-256, and an explicit
redistribution decision before it is added.
